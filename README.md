# xColors Generator

## Description

A random color generator application that fetches beautiful random colors from the xColors API. Users can generate multiple colors at once and view them in an attractive card layout. Each color card displays the color preview along with HEX, RGB, and HSL values. You can copy color values to clipboard and set any color as your page background.

## Features

- Generate random colors from xColors API
- Generate 1 to 20 colors at once
- Display colors in responsive grid layout
- Show HEX, RGB, and HSL color values
- Copy color values to clipboard
- Set any color as page background
- Loading states and error handling
- Responsive design for all screen sizes

## How to Run

1. Open the 'xcolors' folder
2. Double-click on 'index.html' file
3. The color generator will open in your default web browser
4. Enter a number between 1 and 20 in the input field
5. Click Generate button to fetch random colors

## Files

- index.html - Main HTML structure
- style.css - Styling and layout
- script.js - Color fetching and display functionality

## Usage

- Enter the number of colors you want to generate (1-20)
- Click Generate button to fetch colors
- Click Copy button on any color value to copy it to clipboard
- Click Set as Background button to change the page background color
- The application automatically generates one color when the page loads

## API Source

This application uses the xColors API from x-colors.yurace.pro
